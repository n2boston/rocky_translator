Inspired by Grace's translator for Rocky in Project Hail Mary.  

Since I don't have a Rocky to speak in musical tones, for now 
it'll ask you to provide "musical notes" as series of numbers, 
separated by spaces (e.g., 0 1 2 3 4).

If you enter 0 1 2 3 4, it will say in Rocky-like voice "Grace and Rocky save stars" 
for example. 

One day, I will update the script to convert actual musical tones to words, 
but I need to learn more about how music works and learn to play 
something before I can do it.

How the program works:

If an integer is in the dictionary, it speaks out the word in Rocky's voice and prints it out in the terminal.

If any integer is not in the dictionary, it will ask user for defintion to update the dictionary.
