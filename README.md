Inspired by Grace's translator for Rocky in Project Hail Mary (movie and book).

Since I don't have an actual Rocky to speak in musical tones, the program will ask you to provide "musical notes" as series of numbers separated by spaces (e.g., 0 1 2 3 4).

If you enter 0 1 2 3 4, for example, it will say in Rocky-like voice "Grace and Rocky save stars", 6 6 6 will make Rocky say "bad bad bad", 5 5 5 will say "amaze amaze amaze", etc.

One day, I will change the script to convert actual musical tones to words, but I need to learn more about how music works and learn to play something before I can do it.

How the program works:

If an integer is in the dictionary, it speaks out the word in Rocky's voice and prints it out in the terminal.

If any integer is not in the dictionary, it will ask user for defintion to add it to the dictionary.

If you intiate with anything other than the default CSV name, you can train a library from scratch.  It could be useful if you want to extend the Translator class to incorporate actual music notes rather than integers.
